# Shinobi Trial — Quiz Naruto 🍃

Yon ti jwèt quiz sou Naruto, dirèkteman nan navigatè w, san okenn enstalasyon.

## Kijan jwèt la mache

- Gen yon bank de **200 kesyon** sou linivè Naruto (pèsonaj, klan, jutsu, vilaj, Akatsuki, elatriye).
- Chak fwa ou jwe, **15 kesyon** chwazi o aza nan bank lan.
- Ou **genyen** epreuv lan si w reponn **9 oswa plis** kòrèkteman.
- Ou **pèdi** si w gen **8 oswa pi piti** bon repons.
- Ou ka retante otan fwa ou vle — kesyon yo melanje chak fwa.

## Jwe sou entènèt

Si Pages aktive sou repo sa, ou ka jwe dirèkteman sou:

```
https://<non-itilizatè>.github.io
```

(ranplase `<non-itilizatè>` ak vrè non GitHub ou a, oswa ak non repo a si l pa nan fòma `username.github.io`)

## Jwe lokalman

Telechaje `index.html` epi louvri l nan nenpòt navigatè (Chrome, Safari, Firefox, elatriye). Pa gen okenn lòt enstalasyon ki nesesè.

## Fè chanjman

Tout kòd la (HTML, CSS, JavaScript) ann nan yon sèl fichye, `index.html`. Bank kesyon yo nan yon varyab JavaScript ki rele `QUESTIONS` — ou ka ajoute, retire, oswa modifye kesyon ladan l dirèkteman.
