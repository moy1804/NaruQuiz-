## 🎮 À propos

NaruQuiz est une application web inspirée de l'univers Naruto.

Chaque partie sélectionne 15 questions aléatoires parmi une base de 200 questions.

Le joueur doit obtenir au moins 9 bonnes réponses pour gagner.

---
